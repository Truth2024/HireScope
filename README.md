# 🎯 HireScope — Современная платформа для рекрутинга

**HireScope** — это решение для поиска талантов и управления процессом найма. Платформа объединяет в себе интеллектуальный подбор и социальные механики для эффективного взаимодействия между HR и кандидатами.

> **Находите таланты. Отслеживайте прогресс. Нанимайте эффективнее.**

---

## ✨ Основные функции

Платформа разделена на два типа профилей: **HR** (для поиска и управления вакансиями) и **Кандидаты** (для поиска работы и демонстрации навыков).

### 🔐 Безопасность

- Система авторизации на базе **Access & Refresh tokens**
- Защищенные маршруты и middleware
- Автоматическое обновление токенов

### 🔔 Система уведомлений (Real-time & Offline)

- **Мгновенные уведомления** через Pusher при:
  - Отклике кандидата на вакансию (HR получает уведомление)
  - Принятии кандидата (кандидат получает уведомление)
  - Отклонении кандидата (кандидат получает уведомление)
- **Оффлайн-доставка** — уведомления сохраняются в БД
- **Счетчик непрочитанных** уведомлений в реальном времени
- **Кнопка "Отметить все"** для массового прочтения

### 🕵️ Поиск и фильтрация

- Поиск кандидатов и вакансий по ключевым словам
- Фильтрация по навыкам (мультиселект)
- Фильтр по наличию опыта
- Сортировка по:
  - Дате (новые/старые)
  - Рейтингу вакансий
  - Совпадению навыков
- Встроенная пагинация с синхронизацией URL

### 💬 Социальное взаимодействие

- **Бесконечный скролл** комментариев к вакансиям
- Система рейтинга вакансий (1-5 звезд)
- Комментарии с привязкой к пользователям
- Детальная статистика распределения оценок

### 📸 Recruiting Stories

- Лента новостей в формате историй (как в Instagram)
- Прогресс-бар для навигации между историями
- Автоматическая смена историй

### 👤 Управление профилем кандидата

- **Настройка навыков** — интерактивный выбор из предложенных
- **Управление опытом работы**:
  - Добавление/удаление мест работы
  - Компания, должность, количество лет
- **Загрузка аватара**:
  - Автоматическое создание blur-версии
  - Валидация размера и формата
  - Удаление старого аватара при загрузке нового

### 📝 Управление вакансиями (HR)

- **Создание вакансии** с полями:
  - Название, описание, требования
  - Компания, зарплатная вилка
- **Редактирование вакансии**
- **Просмотр откликнувшихся кандидатов**
- **Принятие/отклонение кандидатов** с автоматическими уведомлениями

### 🛡️ AuthWall

- Автоматическое размытие (блюр) аватарок для неавторизованных посетителей
- Защита контента в зависимости от роли
- Редиректы для защищенных маршрутов

### 🌍 Интернационализация

- Полная поддержка двух языков (**RU / EN**)
- Переводы для:
  - Интерфейса
  - Сообщений об ошибках
  - Уведомлений
  - Валидации форм
- SEO-метаданные для каждого языка

### ⚡ Производительность

- Серверный рендеринг (SSR) для быстрого первого отображения
- Предзагрузка данных (prefetch) для плавной навигации
- Кэширование запросов через React Query

---

## 🛠️ Технологический стек

- **Frontend:** Next.js, TypeScript, MobX, TailwindCSS
- **Backend:** Next.js API Routes, MongoDB, Mongoose
- **Реальное время:** Pusher
- **Формы и валидация:** Zod
- **Интернационализация:** next-intl
- **Управление состоянием:** MobX + React Query
- **Файлы:** Dropbox API для хранения фотографий
- **UI библиотека:** MUI + кастомные компоненты

- ✅ **Real-time взаимодействие** — мгновенные уведомления о событиях
- ✅ **Двойная система уведомлений** — Pusher + БД для надежности
- ✅ **Интуитивный интерфейс** — понятный для всех ролей
- ✅ **Безопасность** — JWT токены, защита маршрутов
